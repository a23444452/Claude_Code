# YOLO Learning Assistant Plugin

## Description
YOLO 學習助手，互動式教學指南，解釋 YOLO 概念、架構和最佳實踐。

## Topics Covered

### YOLO 基礎
- 什麼是 YOLO？
- 單階段 vs 兩階段偵測
- YOLO 架構演進 (v1-v11)

### 訓練概念
- Anchor boxes
- Loss function
- Data augmentation
- Transfer learning

### 效能指標
- mAP (Mean Average Precision)
- IoU (Intersection over Union)
- Precision & Recall
- F1-Score

### 最佳實踐
- 資料集準備
- 超參數調整
- 模型選擇
- 部署策略

## Interactive Learning
詢問任何 YOLO 相關問題，獲得詳細解釋和範例！
