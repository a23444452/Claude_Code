# 快速推論測試

使用訓練好的模型對單張圖片進行快速推論測試。

## 執行步驟

1. 檢查模型是否存在
2. 讀取測試圖片
3. 執行推論（使用 YOLO CLI 或 API）
4. 顯示偵測結果：
   - 偵測到的物件數量
   - 各物件的類別和信心度
   - 邊界框座標
5. 儲存標註後的圖片（如果需要）

## 使用範例

```bash
# 使用預設測試圖片
/quick-predict

# 指定圖片路徑
/quick-predict --image dataset/test/sample.jpg

# 調整信心度閾值
/quick-predict --image test.jpg --conf 0.3
```

## 輸出

```
偵測結果：
- 類別 0 (object): 85.2%
  位置: (100, 200) - (150, 250)
- 類別 1 (object): 72.8%
  位置: (300, 400) - (350, 450)

總計：偵測到 2 個物件
推論時間：98ms
```

標註後的圖片儲存在：runs/predict/exp/
